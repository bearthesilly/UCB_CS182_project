# Readme

This is the code for reproducing toy experiment result. This experiment can easily reproduce  obvious Catastrophic Forgetting Phenomenon which is helpful for a close observation to this phenomenon.

How to reproduce this result?

1. run `./download_assets.py` to download the `ag_news` dataset and `distilbert-base-uncased` model
2. run `./main_experiment.py` to start running. The figures of loss and accuracy will be plotted for you.
